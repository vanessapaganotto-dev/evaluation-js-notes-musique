# evaluation-js-notes-musique


Conversion de Notes Musicales (Classique → Américaine)
Description

Ce projet propose une interface web simple permettant de convertir des notes de musique de la notation classique française (do, ré, mi, etc.) vers la notation américaine (C, D, E, etc.).

Le formulaire est créé et injecté dynamiquement dans la page via JavaScript. Lorsqu'une note est sélectionnée, la correspondance américaine s’affiche instantanément sans rechargement.

Fonctionnalités

Génération dynamique du formulaire de sélection via JavaScript

Correspondance entre notation classique et notation américaine

Affichage immédiat du résultat sous le formulaire

Interface légère, sans besoin de backend

Contenu du projet

index.html : page HTML minimale, le contenu est injecté par JavaScript

script.js : script JavaScript qui crée le formulaire, gère la sélection et affiche la correspondance

style.css : styles optionnels pour la mise en forme (non fournis ici)

Installation et utilisation

Copier les fichiers dans un dossier accessible par un serveur web ou ouvrir directement index.html dans un navigateur moderne.

La page affichera un formulaire de sélection de notes musicales.

Choisir une note dans la liste pour voir sa correspondance en notation américaine affichée immédiatement.
